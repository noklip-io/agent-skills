# shadcn-base Skill

shadcn/ui Base UI edition: copy-and-own components built on Base UI primitives with render-based composition.

## Installation

```bash
npx skills add noklip-io/agent-skills --skill shadcn-base
```

## What's Covered

This skill mirrors the Base UI edition of shadcn/ui and organizes its docs for agent use:

- Base UI component index (converted from shadcn docs paths)
- Installation and CLI workflows
- components.json schema and registry config
- Theming and dark mode guidance
- Forms field patterns and form library integrations
- Blocks and MCP usage
- Composition paradigm: render/useRender only (no Radix asChild)
- Examples for complex components and edge cases
- Changelog tracking and issue escalation paths

## Triggers

This skill activates when you mention:

- shadcn/ui Base UI components or "Base UI version of shadcn"
- Base UI primitives with shadcn/ui styling
- render/useRender composition, compound components, CVA variants
- shadcn CLI, components.json, registries, or blocks
- Migrating from Radix shadcn to Base UI shadcn
