# TypeScript

## Core Types
- Components expose `Props` and `State` namespaces for type-safe composition.
- Change handlers use `BaseUIChangeEventDetails<Reason>` and `BaseUIChangeEventReason` types.

## Actions and Refs
- Some components expose `actionsRef` and related action types.

## useRender Types
- Use `ComponentProps` and `ElementProps` from `useRender` to type custom render components.
