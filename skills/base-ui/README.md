# Base UI Skill

Reference and workflows for @base-ui/react (unstyled, accessible React components).

## Installation

```bash
npx skills add noklip-io/agent-skills --skill base-ui
```

## What's Covered

This skill includes 12 focused reference files:

| Reference | Topics |
|-----------|--------|
| `overview.md` | Install, portals, iOS Safari notes, LLM docs access |
| `components.md` | Component index and doc links |
| `styling.md` | className/state, data attributes, CSS variables, styling patterns |
| `composition.md` | Render props, ref forwarding, nested render patterns |
| `customization.md` | Events, eventDetails, allow/stop propagation, controlled state |
| `animation.md` | Data attributes, transitions, Motion, keepMounted, getAnimations |
| `forms.md` | Form/Field patterns, validation, server errors, RHF/TanStack |
| `typescript.md` | Props/State namespaces, event types, actionsRef typing |
| `utils.md` | CSP/Direction providers, mergeProps, useRender patterns |
| `accessibility.md` | ARIA guidance and a11y expectations |
| `edge-cases.md` | Common pitfalls and fixes |
| `examples.md` | Concise runnable examples |

## Triggers

This skill activates when you mention:

- Base UI, @base-ui/react, MUI Base UI
- Base UI components (Dialog, Select, Menu, Popover, Tooltip, etc.)
- render prop composition, useRender, slots
- eventDetails, preventBaseUIHandler, allowPropagation
- portal isolation, keepMounted, data attributes
- Base UI forms, Field/Form, constraint validation
